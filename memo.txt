git 버전확인
git --version

프로젝트 디렉토리에 로컬 저장소를 생성 
git init

git 버전 관리할 파일을 선택(stage에 올림)
git add test.py

하나의 버전으로 만들기
git commit -m "test.py을 커밋"

로그 확인하기
git log

원격 저장소에서 변경된 사항 가져오기 
git fatch origin (가져오기만)
git pull origin (가져와서 merge)

.gitignore
깃이 추적하지 않아야 할 파일을 지정하는 파일 
예) 
파일이름.확장명
*.확장명
폴더명/

branch
작업에 대한 가지
큰 줄기에 영향을 주지 않고 가지에서만 작업을 할 수 있음
각 가지는 자체 커밋 및 변경을 가질 수 있음
> main에 영향을 미치지 않고 여러 기능이나 버그 수정을 동시에 수행할 수 있음

git status
git branch

애플 생성하고 이동 
git checkout -b apple
